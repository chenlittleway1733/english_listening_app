# 英文聽力練習系統 v18

## v18 更新

- 保留 Shopping 1–500 句。
- 保留 Restaurant 1–500 句。
- 保留 School 1–500 句，長度規則與 Shopping 相同。
- 保留出題範圍：`1-100`、`101-200`、`201-300`、`301-400`、`401-500`、`1-500`、`錯題出題`。
- 播放方式改成：按一次播放鈕，句子會播放兩次：
  1. 第一次：女聲
  2. 第二次：男聲
- 移除「播放速度」下拉選項；播放器本身已有速度控制時，可直接用播放器調整。
- 音檔仍產生在系統暫存資料夾 `/tmp`，不使用專案中的 `audio/` 資料夾。
- 主要使用 `edge-tts` 產生女聲與男聲；若雲端暫時無法使用，會自動退回 `gTTS`，讓系統仍可作答。

## 安裝

```bash
pip install -r requirements.txt
```

## 執行

```bash
streamlit run app.py
```
